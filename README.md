# FoodNStuff — Meal Planner

Personal meal prep planner. Calories from the German BLS 4.0 food database.

## Quick Start

```bash
# One-time setup (first run only)
pip install openpyxl
python scripts/convert_bls.py

# Start the app
cd FoodNStuff
python -m http.server 8000
```

Open **http://localhost:8000/web/**

---

## Adding a Meal

Create a `.yaml` file in the right folder:

```
data/meals/
├── breakfast/
├── lunch/
├── dinner/
└── snacks/
```

Meal YAML format:

```yaml
name: My Meal
description: Short description for the card view.
ingredients:
  - name: Hähnchen Brustfilet
    bls_code: V416100
    amount_g: 150
  - name: Reis
    bls_code: C352000
    amount_g: 100
steps:
  - Cook rice
  - Grill chicken
notes: Optional notes shown at the bottom.
```

Refresh the browser — the meal appears automatically.

### Finding BLS Codes

Search [data/ingredients/bls.json](data/ingredients/bls.json) for the ingredient name to get its `bls_code`.
All weights are **raw/uncooked** unless noted otherwise.

## Structure

```
FoodNStuff/
├── BLS_4_0_2025_DE/      ← source database (do not edit)
├── data/
│   ├── meals/            ← your meal YAML files
│   └── ingredients/
│       └── bls.json      ← generated by convert_bls.py
├── scripts/
│   └── convert_bls.py   ← run once to generate bls.json
└── web/                 ← website (AI-managed)
```
